# Java REST Examples

To run a Java REST sample file, do the following:

## Windows
From the src directory:

  - Compile: `javac -classpath ".;../lib/*" JavaFile.java`
  - Run: `java -classpath ".;../lib/*" JavaFile`
  - Replace `JavaFile` with the example you would like to run
  
## Linux / OS X
From the src directory:

  - Compile: `javac -classpath ".:../lib/*" JavaFile.java`
  - Run: `java -classpath ".:../lib/*" JavaFile`
  - Replace `JavaFile` with the example you would like to run
